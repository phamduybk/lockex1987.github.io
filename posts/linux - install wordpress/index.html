Cài đặt WordPress

Vào trang sau để download. Chú ý là .org chứ không phải .com.

https://wordpress.org/download/

Giải nén, copy thư mục wordpress vào thư mục web của Apache (/var/www/html)

Tạo CSDL wordpress trên MySQL trước

create database wordpress;

Truy cập trang localhost/wordpress và làm theo hướng dẫn.
