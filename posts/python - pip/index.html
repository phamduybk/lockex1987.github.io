Python PIP

pip là một hệ thống quản trị package của Python. pip thường được cài đặt sẵn cùng Python.

Sử dụng

Kiểm tra version của pip và cả Python:

pip --version

IMAGE

Tìm kiếm package nào đó, ví dụ Flask:

pip search <some package>

pip search Flask

Cài đặt (và nâng cấp) package

pip install some-package

pip install --user requests

pip install Flask

The --user flag means pip has made Requests available to us, but not to other users.

Cài đặt các gói được khai báo ở file requirements.txt:

pip install -r requirements.txtCài đặt cho phiên bản cụ thể của Python, trong đó ${version} được thay thế bởi 2, 3, 3.4:

pip${version} install some-package

Upgrade pip

sudo pip install --upgrade pip

Upgrade một package nào đó:

pip install --upgrade <some-package>

Hiển thị thông tin của package đã cài đặt

pip show Flask

Hiển thị tất cả các package đã cài đặt

pip freeze

pip list

Có thể dùng tùy chọn --outdated để liệt kê các package đã có phiên bản mới.

Loại bỏ package

pip uninstall some-package

pip uninstall Flask

Thêm tùy chọn proxy

--proxy <proxy>

Specify a proxy in the form [user:passwd@]proxy.server:port.
