Cách mà Elasticsearch hỗ trợ luôn:

https://www.elastic.co/guide/en/elasticsearch/reference/current/modules-snapshots.html

Cách này có vẻ phức tạp.

Các khác đơn giản hơn là sử dụng tool sau:

https://github.com/taskrabbit/elasticsearch-dump

Ví dụ, để export:

elasticdump --input=http://10.30.153.186:9200/voice2text_bca --output=D:\home\voice2text_bca.json

Để import:

elasticdump --output=http://localhost:9200/voice2text_bca --input=D:\home\voice2text_bca.json

File dữ liệu mẫu ở đây.
