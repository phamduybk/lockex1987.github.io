jqm-multiple-month-picker
=========================

Usage
-----

HTML
```
<div id="testarea"></div>
```

Javascript
```
$(document).ready(function() {
  $('#testarea').mmp();
});
```

Retrieve selected values:
```
$('#testarea').mmp('value');
```

Set values:
```
$('#testarea').mmp('value', ['2014-01', '2014-02', '2014-07']);
```
