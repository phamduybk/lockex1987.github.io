Mệnh đề if trong Python

Trong Java, chúng ta có cách viết ngắn gọn sau:

variable = condition ? value_1 : value_2

Trong Python cũng có kiểu viết tương tự vậy, nhưng hơi khác:

variable = value_1 if condition else value_2

Ví dụ:

def demo_if(condition, value_1, value_2):
variable = value_1 if condition else value_2
print(variable)


demo_if(True, "Python", "Java")
demo_if(False, "Huyen", "Duong")
