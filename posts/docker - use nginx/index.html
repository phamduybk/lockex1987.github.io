Sử dụng nginx với Docker

Pull image 

docker pull nginx

Chạy:

docker run nginx

Khi đó truy cập http://172.17.0.2/.

Expose port:

docker run -p 8080:80 nginx

Khi đó truy cập http://172.17.0.2/ hoặc http://localhost:8080/.

Volume:

docker run -v /data/docker:/usr/share/nginx/html -p 80:80 nginx
