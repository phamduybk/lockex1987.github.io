Ứng dụng SSO Admin

Là trang admin để quản trị dữ liệu của dự án SSO.

Cũng là một trang web mẫu để demo các chức năng thêm - xóa - sửa.

Sử dụng các công nghệ phổ biến, dễ dùng như:

  • Bootstrap
  • jQuery
  • common-validation
  • noti
  • Laravel

Không dùng VueJS, không phải là ứng dụng Single Page App.
