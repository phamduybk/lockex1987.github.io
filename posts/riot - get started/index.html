Riot là gì

Riot là một thư viện JavaScript giúp bạn thiết kế trang web của bạn theo hướng component một cách đơn giản và tao nhã. Nó tương tự như Vue component, tuy nhiên đơn giản hơn rất nhiều.

Tại sao lại là Riot

Tại sao bạn lại dùng Riot trong rất nhiều các thư viện JS đã có ngoài kia.

  • Sử dụng đơn giản. Có thể chạy trực tiếp trên trình duyệt luôn, không cần dùng npm compile, không cần học thêm tool như Webpack. Hãy trả lại sự đơn giản cho web (học npm, cli tool như vue-cli, đóng gói như Webpack)
  • Ít các khái niệm mới cần phải học thêm, không phải học về state, vuex, Flux, Redux,...
  • Không cần tải hàng đống node_modules với hàng trăm nghìn file
  • Nhẹ và nhanh
  • Star trên GitHub cao; dù chưa bằng React, Vue, Angular, nhưng hơn những cái khác
  • Hướng component

Tư tưởng là "less is more", "minimalism", KISS.

Riot đơn giản thật nhưng nó cũng rất đầy đủ.

Chúng ta không phải chạy theo các framework. Hãy học thật tốt một cái. Tôi chọn Riot thay cho Vue.

Component-based

Hướng component là một cách tiếp cận khác rất hay trong thiết kế phần mềm. Nó cũng là chia để trị như mô hình MVC, nhưng nếu mô hình MVC chia theo chiều ngang thì component-based sẽ chia theo chiều dọc.

Ví dụ về mô hình MVC ở Angular JS:

  • controllers
    • Features1Controller.js
    • Features2Controller.js
    • ...
  • partials
    • view1.html
    • view2.html
    • ...

Ví dụ về component-based của Riot:

  • features1.tag (bên trong bao gồm HTML, JavaScript, CSS của chức năng 1)
  • features2.tag (bên trong bao gồm HTML, JavaScript, CSS của chức năng 2)
  • ...

Bắt đầu

Cấu trúc của một trang web sử dụng Riot thường như sau:

<!DOCTYPE html>
<html>
<body>
    <!-- Khai báo các tag mới -->
    <mytag></mytag>

    <!-- Định nghĩa các tag mới -->
    <script data-src="mytag.tag" type="riot/tag"></script>
	
    <!-- Thư viện Riot -->
    <script src="https://cdnjs.cloudflare.com/ajax/libs/riot/3.13.2/riot+compiler.min.js"></script>
    
    <!-- Mount để bắt đầu -->
    <script> riot.mount('*'); </script>
</body>
</html>

Chú ý:

  • Chúng ta ánh xạ tới file tag bằng thẻ script, nhưng dùng data-src thay cho src và type là riot/tag
  • Chúng ta có thể sử dụng inline tag nhưng không nên, hãy tách thành file tag riêng
  • Chúng ta có thẻ dùng riot+compiler.js mà không sợ ảnh hưởng hiệu năng, bước compiler diễn ra rất nhanh
  • Chúng ta có thể sử dụng CDN https://cdnjs.cloudflare.com/ajax/libs/riot/3.13.2/riot+compiler.min.js
  • Riot tag trong trong body của trang cần được đóng bình thường: <todo></todo>. Không hỗ trợ tự đóng <todo />.
