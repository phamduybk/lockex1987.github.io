Cấu trúc thư mục của SPA

Không có gì là bắt buộc cả, nhưng nếu có một convention thống nhất thì sẽ tốt hơn (tốt hơn nhiều đấy). Đây là cấu trúc theo convention của tôi (sau khi tham khảo trên mạng và các dự án khác).

  • index.html
  • css
    • Các file CSS riêng của dự án
  • less
    • Tách riêng thư mục less
  • images
  • js
    • Các file JS riêng của dự án
    • app.js
  • libs
    • jquery
      • js
        • jquery.min.js
    • w2ui
      • js
        • w2ui.min.js
      • css
        • w2ui.min.css
  • controllers (nếu nhiều quá thì cũng chia theo module)
    • CategoryController.js
  • directives
  • services
  • partials
    • category.html
  • data
    • Các file JSON dữ liệu
  • media
    • Các file audio, video
  • package.json
  • README.md
  • gulpfile.js
