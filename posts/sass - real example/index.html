1. Download bootstrap css v3.3.7

Các bạn theo đường link này http://getbootstrap.com/docs/3.3/getting-started/#downloadhoặc download tại đây
Đầu tiên để giúp các bạn tiện theo dõi mình sẽ tạo 1 folder Sass_P7 ở ngay desktop, sau đó copy file zip bootstrap đã tải ở trên vào bên trong folder này và giải nén.
Bạn sẽ nhìn thấy cấu trúc tương tự như thế này.

Tạo thêm folder images để chứa ảnh, tạo file index.html là trang mà chúng ta cần làm.

3. Tạo file _colors.scss

Các bạn vào trong folder css và tạo file _colors.scss để lưu các biến màu sắc. Hiện giờ mới chỉ có 1 biến màu trắng, sau này có nhiều màu khác thì các bạn viết thêm vào trong file này 

$white: #fff;

4. Tạo file _mixins.scss

Các bạn vào trong folder css và tạo file _mixins.scss để lưu các mixin. Hiện giờ cũng mới chỉ có 1 mixin Font 

@mixin Font($fontSize, $color, $fontFamily: null, $fontWeight: null){
    font-size: $fontSize;
    color: $color;
    font-family: $fontFamily; 
    font-weight: $fontWeight; 
}

5. Tạo file _fonts.scss

Các bạn vào trong folder css và tạo file _fonts.scss để lưu các font sẽ add vào trang web.
Ở đây mình đã download về font Jura và cách add font mình sử dụng là qua font-face 

@font-face {
    font-family: JuraLight;
    src: url(../fonts/Jura/Jura-Light.ttf);
}
@font-face {
    font-family: JuraRegular;
    src: url(../fonts/Jura/Jura-Regular.ttf);
}
@font-face {
    font-family: JuraBold;
    src: url(../fonts/Jura/Jura-Bold.ttf);
}

5. Tạo file style.scss

Các bạn vào trong folder css và tạo file style.scss và import 3 file scss đã tạo ở trên 

@import "fonts";
@import "colors";
@import "mixins";

8. Viết code file style.scss

Đầu tiên ta đặt ảnh nền cho full toàn bộ trang web ở đây mình có dùng 2 thuộc tính css3 mới là 100vh với 100vw 

@import "fonts";
@import "colors";
@import "mixins";

.wrapper {
    width: 100vw;
    height: 100vh;
    background: url(../images/bg.jpg)no-repeat;
    display: table;
    position: relative;
}

Giao diện như này nhé


Giờ mình style class mask để tạo lớp phủ màu nâu nâu nhìn xuyên qua, cũng tạo cảm giác ánh nắng ấm áp ^^ 

.mask {
    position: absolute;
    top: 0;
    left: 0;
    z-index: 1;
    width: 100%;
    height: 100%;
    background: rgba(233, 155, 75, 0.6);
}

Giao diện như này nhé

Giờ việc tiếp theo là mình căn cho toàn bộ nội dung trang web nằm giữa theo trục dọc OY, sử dụng bằng vertical-align: middle; đồng thời cũng để cho nó nổi lên trên lớp mask nữa 

.tableCell {
    display: table-cell;
    vertical-align: middle;
    position: relative;
    z-index: 2;
}

Giao diện như này nhé


Giờ việc tiếp theo là style các phần logo và chữ của trang web
Các bạn lưu ý là mình đã tạo 1 mixin Font ở đây giờ là lúc sử dụng mixin này kết hợp với font mình đã add 

.logo-text {
    @include Font(70px,$white,JuraLight);
    margin-bottom: 80px;
}
.big-text {
    @include Font(65px,$white,JuraBold);
}
.small-text {
    @include Font(25px,$white,JuraRegular);
    margin-bottom: 50px;
}
