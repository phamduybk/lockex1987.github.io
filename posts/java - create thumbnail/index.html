Use ffmpeg command. You can write Java code to wrap it. Use the right tool to the right job.

For extracting images from a video:

ffmpeg -loglevel error -ss <DURATION> -i <INPUT VIDEO> -vframes 1 -f image2 -s <WxH> <OUTPUT IMAGE>

Example:

ffmpeg -loglevel error -ss 00:05:01 -i 42.mp4 -vframes 1 -f image2 -s 400x300 42.jpg

Explain:

  • -ss duration
    There are two accepted syntaxes for expressing time duration: [-][HH:]MM:SS[.m...] or [-]S+[.m...]
  • -i file
    Input file name
  • -vframes number
    Set the number of video frames to output
  • -f format
    The format is normally auto detected for input files and guessed from the file extension for output files, so this option is not needed in most cases
  • -s size
    Set frame size.
    The format is ‘wxh’ (default - same as source).

Notes

Easy set the PATH variable contain the ffmpeg executable file.
