Nối file video bằng Avidemux
Nối file để dễ dàng lưu trữ cũng như sử dụng.
Công cụ cần thiết là Avidemux. Các bạn cũng có thể sử dụng nhiều chương trình chuyên nghiệp khác, nhưng ở đây mình hướng dẫn sử dụng Avidemux vì nó dễ sử dụng, tiện lợi, nhanh chóng và hỗ trợ rất nhiều định dạng video khác nhau (avi, mkv, mp4, flv, ...).
Ok bắt đầu. Lấy ví dụ mình có nhiêu đây video của Beatles Code


Khá là nhiều đúng không nào. Bây giờ bắt đầu nối lại nhé! Các bạn download Avidemux về cài đặt bình thường. Mở nó lên sẽ có giao diện như thế này:


Trước khi thực hiện, bạn cần xác định chắc chắn rằng các file cần nối phải đạt đủ yêu cầu là cùng có chung các thông số của video (frame width, frame height, bit rate, frame rate, ...). Nếu không đảm bảo được các yếu tố này thì khi nối lại video sẽ bị hình trước tiếng, tiếng trước hình, rè rè, giựt giựt...vân vân!!!

 Bước 1: Bạn vào File > Open để mở file bắt đầu của đoạn video cần nối. Ví dụ như mình cần nối tập 2 của BC thì mình open file Ep 2-1. Sẽ có 1 cửa sổ như sau xuất hiện:


Các bạn chọn Yes để tiếp tục (nếu không có cửa sổ này xuất hiện thì đến bước tiếp theo)

Bước 2: Bạn vào File > chọn Append… (phím tắt: Ctrl + Alt + A), rồi tìm mở tập tin thứ 2 (đối với mình là file Ep 2-2), và tương tự cho các đoạn video tiếp theo (Ep 2-3, Ep 2-4)


Đây là số frame khi mình mở 1 file đầu tiên (20453)


Ở file tiếp theo frame sẽ là 40429 ... và tiếp tục cho đến khi hết file để nối tiếp. Nếu số frame chưa tăng nghĩa là bước thứ 2 của bạn làm sai!


Đây là sau khi mở đầy đủ 4 file, frame là 82659. Bạn chú ý chọn định dạng file ở ô format thành mkv hoặc định dạng khác tùy ý.

Bước 3: Vào File > Save > Save video để lưu lại.


Chú ý nhớ thêm đuôi .mkv (hoặc file gì mà bạn chọn ở trên) vào cho file, không là không xem được đâu đấy!


Sau khi nối xong sẽ có 1 thông báo như thế này


Ok. Vậy là xong quá trình nối file. Việc cuối cùng chỉ là mở file và thưởng thức!
