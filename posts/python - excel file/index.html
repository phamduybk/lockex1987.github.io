Đọc file Excel bằng Python

Chúng ta sẽ thực hành đọc file Excel bằng thư viện openpyxl trong Python.

Đầu tiên chúng ta phải cài thư viện:

$ pip install --user openpyxl

Chúng ta sẽ có các bước:

  • Tạo đối tượng Workbook từ file
  • Lấy đối tượng Sheet
  • Lấy dữ liệu từng Cell
