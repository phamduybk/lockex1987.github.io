2. Media

Cách viết @media tương tự như viết của CSS thường, dùng khi responsive website:

@media (min-width: 1200px) {
    .col-lg-6 {
        width: 50%;
    }
}

$smallTablet: "(min-width: 480px)";

.content {
    margin-top: 30px;
    margin-bottom: 60px;
    
    @media #{$smallTablet} {
        flex-direction: row;
    }
}
