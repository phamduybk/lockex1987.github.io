Chúng ta nên chia nhỏ các thành phần, micro-service.

Mỗi thành phần nên làm đơn giản nhất có thể, dự vào các micro-framework.

Đơn giản là đẹp

Minimalistic

Ví dụ:

  • Python

    • Flask for Python
  • Java
    • Spark for Java
    • Jooby for Java
    • Javalin for Java
    • Jodd for Java
    • Helidon for Java
    • Pippo for Java
    • Rapidoid for Java
    • Armeria for Java
    • Akka HTTP for Java
    • Ratpack for Java
  • Scala
    • Scalatra for Scala
  • Node
    • Express.js for Node.js
    • Built-in module http
  • PHP
    • Lumen for PHP
    • Silex for PHP
    • File PHP thuần
  • Ruby
    • Camping for Ruby
    • Sinatra for Ruby
  • Kotlin
    • Ktor

Các module có thể viết trên các ngôn ngữ khác nhau.
