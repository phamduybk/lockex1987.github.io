Quy tắc đặt tên CSS

Sử dụng dấu phân cách - (kebab-case), không viết kiểu camelCase.

Quy ước đặt tên BEM (Block Element Modifier)

Block như header, footer, nav, section, form, article

Element là các thành phần nhỏ hơn như logo, button, danh sách.

Modifier là các trạng thái, style.

Ví dụ:

btn--green

info__title

Sử dụng BEM với SASS

Ví dụ

.youtube-header {
  
  &__logo {
    
  }
  &__search {
    
  }
  &__list {
    
  }
  &__item {
    
  }
}
