CSS Selector

Danh sách

Selector Ví dụ Mô tả
.class .intro Các phần tử có class là "intro"
.class1.class2 .name1.name2 Các phần tử có class bao gồm cả "name1" và "name2". Ví dụ:
<div class="name1 name2">...</div>
.class1 .class2 .name1 .name2 Các phần tử có class là "name2" mà là con cháu của một phần tử có class là "name1". Ví dụ:
<div class="name1">
    <div class="name2">
        ...
    </div>
</div>
#id #firstname Các phần tử với id là "firstname"
* * Tất cả các phần tử
element p Các thẻ p
element,element div, p Các thẻ div và thẻ p
element element div p Các thẻ p bên trong thẻ div
element>element div > p Các thẻ p là con trực tiếp của thẻ div
element+element div + p Thẻ p mà được đặt ngay sau thẻ div
element1~element2 p ~ ul Các thẻ ul mà có đằng trước là một thẻ p
[attribute] [target] Các phần tử với thuộc tính target
[attribute=value] [target=_blank] Các phần tử với thuộc tính target bằng "_blank"
[attribute~=value] [title~=flower] Các phần tử với thuộc tính title mà chứa từ "flower"
[attribute|=value] [lang|=en] Các phần tử với thuộc tính lang bắt đầu bằng "en"
[attribute^=value] a[href^="https"] Các phần tử a có thuộc tính href bắt đầu bằng "https"
[attribute$=value] a[href$=".pdf"] Các phần tử a có thuộc tính href kết thúc bằng ".pdf"
[attribute*=value] a[href*="w3schools"] Các phần tử a có thuộc tính href chứa xâu con "w3schools"
:active a:active Các link active
::after p::after Insert something after the content of each <p> element
::before p::before Insert something before the content of each <p> element
:checked input:checked Các phần tử input với trạng thái được check
:default input:default Các phần tử input với trạng thái mặc định
:disabled input:disabled Các phần tử input với trạng thái disable
:empty p:empty Các phần tử p mà không có con (không có các text node)
:enabled input:enabled Các phần tử input với trạng thái enable
:first-child p:first-child Phần tử p mà là con đầu tiên của node cha.
::first-letter p::first-letter Chữ cái đầu tiên của các thẻ p
::first-line p::first-line Dòng đầu tiên của các thẻ p
:first-of-type p:first-of-type Phần tử p mà là phần tử p đầu tiên của node cha
:focus input:focus Các phần tử input mà đang được focus
:hover a:hover Link có chuột ở trên
:in-range input:in-range Selects input elements with a value within a specified range
:indeterminate input:indeterminate Selects input elements that are in an indeterminate state
:invalid input:invalid Selects all input elements with an invalid value
:lang(language) p:lang(it) Selects every <p> element with a lang attribute equal to "it" (Italian)
:last-child p:last-child Selects every <p> element that is the last child of its parent
:last-of-type p:last-of-type Selects every <p> element that is the last <p> element of its parent
:link a:link Selects all unvisited links
:not(selector) :not(p) Selects every element that is not a <p> element
:nth-child(n) p:nth-child(2) Selects every <p> element that is the second child of its parent
:nth-last-child(n) p:nth-last-child(2) Selects every <p> element that is the second child of its parent, counting from the last child
:nth-last-of-type(n) p:nth-last-of-type(2) Selects every <p> element that is the second <p> element of its parent, counting from the last child
:nth-of-type(n) p:nth-of-type(2) Selects every <p> element that is the second <p> element of its parent
:only-of-type p:only-of-type Selects every <p> element that is the only <p> element of its parent
:only-child p:only-child Selects every <p> element that is the only child of its parent
:optional input:optional Selects input elements with no "required" attribute
:out-of-range input:out-of-range Selects input elements with a value outside a specified range
::placeholder input::placeholder Selects input elements with placeholder text
:read-only input:read-only Selects input elements with the "readonly" attribute specified
:read-write input:read-write Selects input elements with the "readonly" attribute NOT specified
:required input:required Selects input elements with the "required" attribute specified
:root :root Selects the document's root element
::selection ::selection Selects the portion of an element that is selected by a user
:target #news:target Selects the current active #news element (clicked on a URL containing that anchor name)
:valid input:valid Selects all input elements with a valid value
:visited a:visited Selects all visited links
