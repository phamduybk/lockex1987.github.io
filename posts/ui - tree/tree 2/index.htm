Nội dung có thể mở rộng

Problem

Bạn muốn trình bày một menu điều hướng mà trông và thực thi như cây tôn ty có thể mở rộng/đóng được trình bày ở frame bên trái của rất nhiều sản phẩm phổ biến (Windows Explorer, Adobe Acrobat Reader).

Solution

Chúng ta cần thiết lập một số style sheet. Đầu tiên, chúng ta cần bỏ các ký tự đầu mỗi mục của bố cục (list-style-type:none). Tiếp theo là thiết lập cho ảnh thể hiện ở class symbols.

<style type="text/css">
#containerul li {
list-style-type:none;
}

#containerul .symbols {
float:left;
width:12px;
height:1em;
background-position:0 50%;
background-repeat:no-repeat;
}
</style>

Chúng ta chỉ nên thêm vào một cách động, không can thiệp trực tiếp. Có nghĩa là code HTML vẫn là các thẻ <ul> và <li> bình thường, không chứa các mã HTML khác.

Ví dụ chúng ta có một menu sau:

<ul id="containerul">
	<li>Blog Sites
	<ul>
		<li><a href="http://www.autisticcuckoo.net">Tommy Olsen</a></li>
		<li><a href="http://www.zeldman.com">Jeffrey Zeldman</a></li>
		<li><a href="http://www.meyerweb.com">Eric Meyer</a></li>
	</ul>
	</li>
	<li>Other Sites
	<ul>
		<li><a href="http://www.accessify.com/">Accessify</a></li>
		<li><a href="http://www.w3.org/">World Wide Web Consortium</a>
		<ul>
			<li><a href="http://validator.w3.org/">The W3C Markup Validator</a></li>
			<li><a href="http://jigsaw.w3.org/css-validator/">The W3C CSS Validator</a></li>
			<li><a href="http://www.w3.org/TR/WAI-WEBCONTENT/">The Web Content Accessibility Guidelines</a></li>
		</ul>
		</li>
	</ul>
	</li>
</ul>

Menu của chúng ta có một định danh riêng, trong ví dụ này là "containerul". Sau khi thành lập menu bằng HTML như trên, chúng ta cần gọi hàm Javascript như sau:

<script type="text/javascript">
createTree("containerul");
</script>

Ta thấy rằng, chúng ta chỉ quan tâm đến các thẻ <li> mà có thẻ <ul> bên trong. Với mỗi thẻ như thế, chúng ta sẽ bao nó bởi một thẻ <span>, có hai trạng thái ("none", "block") tương ứng với thẻ được hiển thị hay không hiển thị.

<script type="text/javascript">
function createTree(idname) {
	var list = document.getElementById(idname).getElementsByTagName("li");
	for (var i = 0; i < list.length; i++) {
		var spanTag = document.createElement("span");
		spanTag.className = "symbols";
		if (list[i].getElementsByTagName("ul").length > 0) {
			spanTag.style.backgroundImage = "url(plus.png)";
			spanTag.onclick = function() {
				toogle(this.parentNode);
			}
			list[i].getElementsByTagName("ul")[0].style.display = "none";
		}
		else {
			spanTag.style.backgroundImage = "url(page.png)";
		}
		list[i].insertBefore(spanTag, list[i].firstChild);
	}
}

function toogle(el) {
	el.getElementsByTagName("ul")[0].style.display = (el.getElementsByTagName("ul")[0].style.display == "block") ? "none" : "block";
	el.getElementsByTagName("span")[0].style.backgroundImage = (el.getElementsByTagName("ul")[0].style.display == "block") ? "url(minus.png)" : "url(plus.png)";
}
</script>

Có thể có nhiều tính năng khác để thêm vào (ví dụ, lưu giữ trạng thái các thẻ mở hay đóng bằng cookie). Tuy nhiên, để giữ tính đơn giản sẽ không trình bày ở đây. Cái này cũng có thể đáp ứng nhu cầu của bạn rồi.

Notes

Các hàm sử dụng: document.createElement, insertBefore, getElementsByTagName, getElementById.

Các thuộc tính: length, className, style.display, style.backgroundImage, onClick.
