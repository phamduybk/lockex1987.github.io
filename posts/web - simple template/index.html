Simple Template

Có 4 cách để xây dựng tầng view của một ứng dụng web dưới dạng template:

  1. Sử dụng các kỹ thuật phía server như include hoặc tiles với ngôn ngữ như PHP, Java
  2. Sử dụng thẻ iframe
  3. Sử dụng thẻ object
  4. Sử dụng AJAX

Cách 1 thì chúng ta phải lập trình phía server một chút. Ngoài ra các trang web sẽ đều là động, do đó có thể không cache được.

Nếu sử dụng thẻ iframe hoặc object thì giao diện outline sẽ xấu. Dù chúng ta có thể bỏ các đường viền nhưng mà còn độ cao thì khó, sẽ có scroll. Với cách dùng iframe, chúng ta có thể sửa một chút, thêm một bước cập nhật nội dung iframe vào một thẻ div khác để giao diện tự động điều chỉnh height.

Chúng ta hãy thử làm với cách thứ 4. Chú ý AJAX không thể chạy trên một file html ở local, phải để vào một server nào đó. Thực ra thì chính trang web bạn đang xem được xây dựng với kỹ thuật này. Chỉ với AJAX, không hề có PHP hay Java nào.
