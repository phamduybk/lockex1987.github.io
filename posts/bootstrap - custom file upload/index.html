Bootstrap Custom File Input

Chúng ta tạo một đối tượng bao với class là custom-file. Bên trong là đối tượng file input với class là custom-file-input và label với class là custom-file-label.

Demo:

Chú ý không hiển thị file đang được chọn là gì.

File input mặc định:

Nếu chỉ là form đơn giản với một trường input file, chúng ta nên sử dụng luôn mặc định. Ở trường input file mặc định có hiển thị file đang được chọn.

Nếu như upload nhiều file, hoặc cần preview, hoặc cần kéo thả thì không nên sử dụng mặc định. Ngoài ra cũng không nên sử dụng Bootstrap vì cũng không đủ chức năng. Hãy tự làm, cũng đơn giản.
