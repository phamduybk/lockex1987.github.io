Bạn phải đánh đi đánh lại các lệnh git add/commit/push.

Đầu tiên tôi định sử dụng alias, nhưng alias không hỗ trợ tham số, mà lệnh git commit lại cần có tham số.

Thêm đoạn sau vào file ~/.bashrc để bạn có thể sử dụng hàm ở các chỗ khác:

git-chain() {
    git add .
    git status
    git commit -m "$1"
    git push
    # sh deploy.sh
}

Sử dụng lệnh sau:

$ git-chain "Test 7"
