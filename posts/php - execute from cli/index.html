PHP CLI

Tôi dự định học lại PHP. Chỉ học những thứ cơ bản thôi:

  • Xử lý mảng, từ điển
  • Xử lý xâu
  • Biểu thức chính quy
  • Hướng đối tượng

Tôi sẽ không học những cái sau, vì nó sẽ được hỗ trợ bởi framework. Chúng ta không nên code PHP thuần mà hãy sử dụng framework cho chuẩn, hỗ trợ nhiều, có nhiều best practice.

  • Thao tác CSDL
  • Thao tác web (lấy thao số, xử lý form, session, MVC,...)

Vì mục đích của tôi chỉ là học PHP cơ bản thôi, thế nên chẳng nhẽ tôi lại phải đẩy lên server (Apache, nginx) và khi muốn test lại phải vào trình duyệt và vào đến file tương ứng.

Tương tự như cách mà tôi học ES6, không cần tạo file HTML và chèn code JavaScript, tôi có thể thực hiện ngay bằng lệnh node.

Tôi sẽ thực hiện các lệnh PHP bằng lệnh php.

Đầu tiên cần đảm bảo bạn cài PHP đúng:

php -v

Giả sử bạn viết code ra file hello.php với nội dung:

<?php
echo 'Hello World';

Thực thi file bằng lệnh sau:

php hello.php

Nội dung in ra sẽ là "Hello World".
