Hướng dẫn AngularJS

AngularJS là một cuộc cách mạng ở thời điểm nó ra đời. Nhiều khái niệm, tư tưởng hoàn toàn mới.

Nó khá là đầy đủ.

  • Two-way binding
  • Condition
  • Repeat
  • Style, class
  • Events
  • Form element
  • Validation
  • Directive (custom tag)
  • Animation
  • Filter
  • AJAX
  • Routing (SPA - Single Page Application)

Giờ AngularJS gần như không được sử dụng nữa.
