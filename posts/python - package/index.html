Package trong Python

Có thể gom nhiều module .py vào một thư mục. Khi đó tên thư mục là tên của package. Chúng ta cũng cần tạo file __init__.py trong thư mục này.

Như vậy cấu trúc của một package sẽ như sau:

|-- mypackage
|    |-- __init__.py
|    |-- mymodule1.py
|    |-- mymodule2.py

Có thể sử dụng mymodule1 theo cú pháp import sau:

import mypackage.mymodule1

hoặc

import mypackage.mymodule1 as mymodule1

hoặc

import mypackage.mymodule1 as mod

hoặc

from mypackage import mymodule1

Khi sử dụng một module thuộc một package thì các lệnh trong file __init__.py sẽ được thực hiện trước. Thông thường thì file __init__.py sẽ rỗng.
