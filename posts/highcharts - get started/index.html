Highcharts - Get started

Khai báo thẻ div chứa đồ thị:

<div id="container" style="min-width: 310px; height: 400px; margin: 0 auto"></div>

Thêm các thư viện cần thiết

<script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>
<script src="https://code.highcharts.com/highcharts.js"></script>
<script src="https://code.highcharts.com/modules/exporting.js"></script>

Script tạo đồ thị:

Cách 1:

new Highcharts.Chart({
    chart: {
        renderTo: 'container'
    },
    ... // other options
});

Cách 2:

Highchart('container', options);

Cách 3:

$('#container').highcharts(options);

Cách 1 và cách 2 thì ta có thể đặt biến đồ thị và thay đổi sau này.

xAxis mà là categories thì series phải có data là mảng có độ dài tương ứng

xAxis mà là datetime thì series có thể có dạng:

{
    data: [29.9, 71.5, 106.4, 129.2, 144.0, 176.0, 135.6, 148.5, 216.4, 194.1, 95.6, 54.4],
    pointStart: Date.UTC(2010, 0, 1),
    pointInterval: 3600 * 1000 // one hour
}
