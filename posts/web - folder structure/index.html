Cấu trúc thư mục của ứng dụng web

Không có gì là bắt buộc cả, nhưng nếu có một convention thống nhất thì sẽ tốt hơn (tốt hơn nhiều đấy).

Đây là cấu trúc theo convention của tôi (sau khi tham khảo trên mạng và các dự án khác).

  • index.html
  • css
    • style.css
  • less
    • style.less
  • sass
    • style.scss
  • images
  • js
    • app.js
  • libs
    • jquery
      • jquery.min.js
    • w2ui
      • js
        • w2ui.min.js
      • css
        • w2ui.min.css
      • fonts
        • abc.ttf
  • fonts
    • abc.ttf
  • controllers (AngularJS, nếu nhiều quá thì cũng chia theo module)
    • CategoryController.js
  • directives (AngularJS)
  • services (AngularJS)
  • partials (AngularJS)
    • category.html
  • components (Riot)
  • data
    • Các file JSON dữ liệu
  • media
    • Các file audio, video
  • upload
    • Các file upload, có thể chia thư mục con
  • package.json
  • README.md
  • gulpfile.js
