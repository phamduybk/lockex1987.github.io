JavaScript Function

JavaScript functions are defined with the function keyword. You can use a function declaration or a function expression.

A function declaration:

function myFunction(a, b) {
  return a * b;
}

A function expression:

var x = function(a, b) {
  return a * b;
}

Functions are Objects

The typeof operator in JavaScript returns "function" for functions. However, JavaScript functions can best be described as objects. JavaScript functions have both properties (arguments) and methods (toString, call, apply).

The toString() method returns the function as a string (function declaration).

The arguments property is an array that contains the actual arguments received when the function was invoked.

function argumentsDemo() {
  var output = '';
  var a = arguments;
  var i;
  for (i = 0; i < a.length; i++) {
    output += a[i] + ', ';
  }
  document.getElementById('argumentsDemoOutput').textContent = output;
}

function getFunctionDeclaration() {
  document.getElementById('getFunctionDeclarationOutput').textContent = argumentsDemo.toString();
}

argumentsDemo(1, 2, 3, 4, 5, 99);

Function Invocation

JavaScript functions can be invoked in four different ways.

  • Invoking a Function as a Function: myFunction(10, 2);
  • Invoking a Function as a Method: myObject.fullName();
  • Invoking a Function with a Function Constructor: var x = new myFunction("John", "Doe");
  • Invoking a Function with a Function Method: myObject = myFunction.call(myObject, 10, 2);
function callDemo(a, b) {
  return a * b;
}

function applyDemo(a, b) {
  return a * b;
}

document.getElementById("callDemoOutput").textContent = callDemo.call(null, 10, 2);
document.getElementById("applyDemoOutput").textContent = callDemo.apply(null, [10, 2]);
