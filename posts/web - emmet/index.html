Emmet

By instantly expanding simple abbreviations into complex code snippets, Emmet can turn you into a more productive developer.

Abbreviations

html:5 or ! for an HTML5 doctype

html:xt for an XHTML transitional doctype

html:4s for an HTML4 strict doctype

tag#id

tag.class

#id

.class

tag{content}

tag[attribute=value]

> Child operator

Eg: p>span

+ Sibling operator

Eg: h1+h2

^ Climb-up operator

Eg: p>span^p

Grouping by parentheses

Eg: (.foo>h1)+(.bar>h2)

* Multiplication operator

Eg: ul>li*3

lorem or ipsum: Lorem ipsum
