Cho vào thẻ div với class là bookmarks.

Thêm CSS:

<style>
.bookmarks a { background-repeat: no-repeat; background-position: left center; padding-left: 30px; line-height: 30px; }
/*.bookmarks a { display: block; }*/
.bookmarks a:after { content: "\a "; white-space: pre; }
</style>

Thêm JavaScript:

<script>
document.querySelectorAll('a').forEach(a => {
    var icon = a.getAttribute('icon');
    a.style.backgroundImage = `url(${icon})`;
});
</script>
