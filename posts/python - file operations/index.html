Thao tác với file trong Python

Để thao tác với file trong Python chúng ta thường sử dụng các module là os, os.path, shutil.

Tạo thư mục

Để tạo thư mục, chúng ta sử dụng hàm os.mkdir hoặc os.makedirs.

Giả sử chúng ta muốn tạo thư mục my_folder, trong folder_3 lại có thư mục con là my_sub_folder. Khi đó nếu chúng ta sử dụng lệnh os.mkdir('my_folder/my_sub_folder') sẽ bị lỗi. Cần sử dụng hàm os.makedirs('my_folder/my_sub_folder') trong trường hợp này.

Xóa thư mục

Để xóa thư mục, chúng ta sử dụng hàm os.rmdir hoặc os.removedirs.

Khi tôi test thì cả hai hàm os.rmdir và os.removedirs chỉ xóa được thư mục rỗng. Để xóa thư mục đã chứa dữ liệu thì chúng ta sử dụng shutil.rmtree.

Liệt kê thư mục

Để liệt kê thư mục, chúng ta sử dụng hàm os.listdir. Hàm sẽ trả về một list str là tên các file hoặc thư mục con.

Đổi tên file

Để đổi tên file hoặc di chuyển file, chúng chúng ta có thể sử dụng hàm os.rename.

Xóa file

Sử dụng hàm os.remove.

Thao tác với đường dẫn

Để thao tác với đường dẫn, chúng ta sử dụng module os.path.
