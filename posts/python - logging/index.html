Xử lý log trong Python

Log rất quan trọng. Nó đóng vai trò như là hộp đen của chương trình của bạn mỗi khi xảy ra sự cố.

Các yêu cầu:

  • Log ra file và console
  • Dễ switch giữa file và console
  • Các log lỗi log ra file riêng
  • Hiển thị rõ file lỗi, dòng lỗi
  • Định dạng: thời gian, tên file, tên hàm, dòng, level, nội dung
  • Setup đơn giản, chỉ 1 chỗ, những chỗ khác dùng bình thường
  • Log của các thư viện cũng được xử lý
  • Tách file log theo ngày
  • In ra cả ký tự tiếng Việt
