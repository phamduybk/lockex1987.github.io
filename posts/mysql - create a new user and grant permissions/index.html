How to create a new user and grant permissions in MySQL

Let's start by making a new user within the MySQL shell:

CREATE USER 'root'@'%' IDENTIFIED BY 'Abcd@1234';

Sadly, at this point, new user has no permissions to do anything with the databases. In fact, if new user even tries to login (with username and password), he/she will not be able to reach the MySQL shell.

Therefore, the first thing to do is to provide the user with access to the information they will need.

GRANT ALL PRIVILEGES ON billy.* TO 'root'@'%';

The asterisk in this command refers to the tables that they can access.

Once you granted permissions to your new user, always be sure to reload all the privileges:

FLUSH PRIVILEGES;

Your changes will now be in effect.

With above commands, you will create an user that can connect to the MySQL server from external computers.
