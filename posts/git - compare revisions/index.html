Xem Lịch Sử

git log

Xem lịch sử các lần commit:

git log

Xem lịch sử các lần commit của 2 commit gần nhất (hoặc một số bất kỳ):

git log -2

Xem lịch sử commit bao gồm cả thông tin các thay đổi:

git log -p

Xem lịch sử commit dưới dạng một dòng cho dễ nhìn:

git log --pretty=oneline

gitk

Bạn có thể sử dụng công cụ đồ họa gitk được xuất bản kèm với git. Nếu bạn gõ gitk trên thư mục của dự án, bạn sẽ thấy giống như hình dưới.

Bạn có thể xem lịch sử commit ở phần nửa trên của cửa sổ cùng cùng một biểu đồ "cây" (ancestry) trực quan. Phần xem diff ở nửa dưới của cửa sổ hiện thị các thay đổi trong bất kỳ commit nào bạn click ở trên.

git diff

Xem thay đổi (chưa đc add) của những file hiện tại:

git diff

Xem thay đổi giữa local mà master:

git diff origin/master

Xem thay đổi (đã được add, chưa commit):

git diff --cached

Xem thay đổi trước khi push:

git diff --cached origin/master

Xem thay đổi giữa 2 commit:

git diff COMMIT1_ID COMMIT2_ID

Xem thay đổi của 1 lần commit (so với trước đó):

git diff COMMIT_ID

Hiển thị các thay đổi từ cách đây 1 commit:

git diff HEAD~1 HEAD

Chỉ hiển thị danh sách file và trạng thái, thêm tùy chọn --name-status, ví dụ:

git diff --name-status HEAD~1 HEAD
