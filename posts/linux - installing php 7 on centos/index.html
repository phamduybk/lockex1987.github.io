Repo php trên CentOS 7 mặc định là PHP 5.4, cái mà đã không còn được cộng đồng lập trình hỗ trợ nữa. Chính vì vậy bài này mình sẽ hướng dẫn các bạn cài đặt PHP 7.0 hoặc version mới hơn trên CentOS 7.

Bước 1:

Để cài PHP 7, đầu tiên cần cài repo EPEL, REMI lên với câu lệnh sau:

# yum install https://dl.fedoraproject.org/pub/epel/epel-release-latest-7.noarch.rpm
# yum install http://rpms.remirepo.net/enterprise/remi-release-7.rpm

Nếu không cài được bằng URL như trên thì có thể download file và cài bằng file.

Bước 2:

Tiếp theo, bạn cần cài phần mềm quản lý tắt bật các repo dưới local mà không cần thao tác bằng tay chỉnh sửa trong các file trong folder "/etc/yum.repos.d/":

# yum install yum-utils

Bước 3:

Bật 7.0 nếu cài đặt PHP 7.0

# yum-config-manager --enable remi-php70

Bật 7.1 nếu muốn cài đặt PHP 7.1:

# yum-config-manager --enable remi-php71

Hoặc cài PHP 7.2:

# yum-config-manager --enable remi-php72

Bước 4:

Cài đặt các module cần thiết bằng câu lệnh:

# yum install php php-mcrypt php-cli php-gd php-curl php-mysql php-ldap php-zip php-fileinfo php-redis

Bước 5:

Bước cuối cùng kiểm tra phiên bản PHP bằng câu lệnh:

# php -v
