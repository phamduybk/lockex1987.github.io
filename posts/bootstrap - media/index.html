Bootstrap Media

Bootstrap provides an easy way to align media objects (like images and videos) to the left or right of some content. This can be used to display blog comments, tweets and so on.

Usage:

  • Use a <div> element with the .media class to create a container for media objects
  • Use the .media-left class to align the media object (image, video) to the left, or the .media-right class to align to the right
  • The media object has .media-object class
  • Text that should appear next to the media is placed inside a container with .media-body class
  • Additionally, you can use .media-heading for headings
  • The media object can also be top, middle or bottom aligned with .media-top, .media-middle, .media-bottom class

Layout:

<div class="media">
    <div class="media-left">
        <img class="media-object" src="..." alt="...">
    </div>
    <div class="media-body">
        <h4 class="media-heading">Media heading</h4>
        <p>lorem...</p>
    </div>
</div>

Notes:

  • Media objects can also be nested
  • You must specify the width of media object
  • .media-body, .media-left, .media-right have display: table-cell
