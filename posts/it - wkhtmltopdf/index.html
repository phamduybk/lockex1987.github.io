wkhtmltopdf (Wk HTML to PDF) và wkhtmltoimage là các tiện ích dòng lệnh để render HTML thành PDF và các định dạng ảnh khác sử dụng engine Qt WebKit. Những tiện ích chạy hoàn toàn "headless" và không yêu cầu màn hình hiển thị.

Để sử dụng, đầu tiên chúng ta phải download bản binary đã dịch hoặc là dịch từ source. Chi tiết ở https://wkhtmltopdf.org/downloads.html.

Sau đây là cách tôi cài đặt trên Linux Mint.

sudo apt-get install xfonts-75dpi
#sudo dpkg -i wkhtmltox-0.12.2.1_linux-trusty-i386.deb
sudo dpkg -i wkhtmltox_0.12.5-1.xenial_amd64.deb

Hai tiện ích sau sẽ được cài đặt:

wkhtmltoimage
wkhtmltopdf

Chạy các lệnh sau:

wkhtmltopdf http://google.com google.pdf
wkhtmltopdf http://wkhtmltopdf.org/ test.pdf
wkhtmltopdf toc https://www.tecmint.com/sudoers-configurations-for-setting-sudo-in-linux/ 10-Sudo-Configurations.pdf
wkhtmltopdf -L 0 -R 0 -T 0 -B 0 --footer-right "Page [page] of [toPage]   " --footer-font-size 8 "index.html" cv.pdf
wkhtmltopdf -L 0 -R 0 -T 0 -B 0 --footer-right "Page [page] of [toPage]   " --footer-font-size 8 https://lockex1987.github.io/posts/it%20-%20my%20cv/dist/index.html cv.pdf
wkhtmltopdf https://lockex1987.github.io/posts/web%20-%20highlightjs/ color.pdf
