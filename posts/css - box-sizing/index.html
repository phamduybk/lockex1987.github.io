CSS3 Box Sizing

The CSS3 box-sizing property allows us to include the padding and border in an element's total width and height.

Without box-sizing

By default, the width and height of an element is calculated like this:

actual width = width + padding + border

actual height = height + padding + border

This means: When you set the width/height of an element, the element often appear bigger than you have set.

The following illustration shows two DIV elements with the same specified width and height:

box-sizing-01.html

The two DIV elements above end up with different sizes (because div2 have a padding specified).

So, for a long time web developers have specified a smaller width value than they wanted, because they had to subtract out the padding and border.

With CSS3, the box-sizing property solves this problem.

With box-sizing

Syntax:

box-sizing: content-box|border-box|initial|inherit;

If you set box-sizing: border-box; on an element, padding and border are included in the width and height:
