Bootstrap

Tôi là một người luôn muốn tự code từ những cái thuần (pure), gốc (core) mà không sử dụng thư viện. Cái đó thì cũng tốt thôi, vì tôi sẽ hiểu được bản chất và sẽ tùy chỉnh được nhiều. Nhưng nếu sử dụng thư viện, chúng ta sẽ an tâm và có thể tập trung được vào "nghiệp vụ". "Nghiệp vụ" và "trải nghiệm" mới là cái quan trọng của một dự án.

Hãy sử dụng library cho các dự án lớn (kể cả nhỏ).

Hãy chọn lựa library:

  • AngularJS: framework, single page application, interactive with server
  • Bootstrap: GUI
  • jQuery: library, interactive with GUI

Library sẽ giúp chúng ta yên tâm hơn là tự làm vì nó đã được test nhiều. Chúng ta có thể tự làm các ví dụ đơn giản để hiểu nguyên lý, nhưng mà hãy sử dụng library.

Library cũng dạy chúng ta cách code chuẩn.

---------------------------------

Bootstrap là một framework miễn phí và mã nguồn mở dùng để thiết kế các trang web. Nó bao gồm các pattern cho typography, form, button, navigation,... và các mở rộng JavaScript khác như carousel.

Bootstrap được phát triển bởi Mark Otto và Jacob Thornton ở Twitter như là một framework để khuyến khích việc consistency giữa các ứng dụng. Trước Bootstrap, nhiều thư viện khác nhau được sử dụng dẫn đến việc inconsistency và khó bảo trì. File CSS có khi hàng chục nghìn dòng.

Bootstrap được release thành mã nguồn mở vào năm 2011.

Năm 2012, Bootstrap 2 được release, thêm hệ thống layout 12 cột responsive, Glyphicon,...

Năm 2013, Bootstrap 3 được release, thiết kế lại các thành phần sử dụng flat design và cách tiếp cận mobile first.

Năm 2014, Mark Otto thông báo đang phát triển Bootstrap 4. Đến thời điểm hiện tại (30/04/2017) thì phiên bản 4 này vẫn đang ở trạng thái alpha.

Sử dụng CDN:

<!-- Latest compiled and minified CSS -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css"/>

<!-- Optional theme -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap-theme.min.css"/>

<!-- Latest compiled and minified JavaScript -->
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js"></script>

Cách an toàn là download nó về local. Khi mất mạng hoặc khi ứng dụng được triển khai nội bộ thì đây là cách bắt buộc.

Chúng ta sẽ sử dụng phiên bản 3.3.7.

File JavaScript của Bootstrap yêu cầu jQuery. Sử dụng jQuery phiên bản 3.2.1.

Chúng ta sẽ có một template như sau:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8"/>
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1"/>
    <title>Bootstrap Basic Template</title>
    
    <!-- jQuery -->
    <script src="lib/jquery/js/jquery.min.js"></script>

    <!-- Bootstrap -->
    <link href="lib/bootstrap/css/bootstrap.min.css" rel="stylesheet"/>
    <script src="lib/bootstrap/js/bootstrap.min.js"></script>
  </head>
    
  <body>
    <h1>Hello, world!</h1>
  </body>
</html>

Bạn có thể vô hiệu hóa tính năng zoom ở các thiết bị di động bằng cách thêm user-scalable=no vào meta viewport. Người dùng khi đó sẽ chỉ có thể cuộn trang, tạo cảm giác người dùng đang sử dụng một ứng dụng điện thoại thuần túy.

<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no">

Typography

Cỡ font mặc định là 14px, line-height là 1.428. Cái này được áp dụng với body và tất cả các đoạn văn (thẻ p).

Ngoài ra, các thẻ p có margin-bottom bằng 1/2 line-height.

Các thẻ h1, h2, h3, h4, h5, h6

small

mark

abbr

blockquote

dl

code

kbd

pre

Chúng ta chỉ sử dụng thẻ, không thêm class nào.

Để đổi màu chữ theo đúng ý nghĩa, chúng ta có các lớp sau: .text-muted, .text-primary, .text-success, .text-info, .text-warning, .text-danger.

Để đổi màu nền, chúng ta sử dụng các lớp .bg-* tương ứng: .bg-primary, .bg-success, .bg-info, .bg-warning, .bg-danger.
