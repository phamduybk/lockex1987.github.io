Disable tính năng tự ghép dòng comment trên Eclipse

Là người thường xuyên sử dụng Eclipse, chắc bạn đã gặp những lúc mình chỉnh code và comment thành các dòng rất đẹp rồi, vậy mà khi nhấn Ctrl + Shift + F để format lại code thì các dòng tự động được ghép, không xuống dòng ở những chỗ mong muốn.

Ví dụ, ta có comment đẹp đẽ như sau:

Comment OK

Sau khi format code thì:

Comment ugly

Chúng ta có thể chỉnh lại bằng cách:

  • vào menu Window/ Preferences
  • vào mục Java/ Code Style/ Formatter
  • nhấn vào Edit... của Active profile
  • đến tab Comments
  • tích chọn Never join lines
Java formatter
Never join comment lines

Giờ bạn có thể format code thoải mái :)
