Thiết lập proxy cho npm

Giả sử máy tính ở công ty của bạn không được tự do thoải mái kết nối Internet nên bạn không thể cài đặt một package NodeJS nào đó qua lệnh npm install bình thường. Hãy thử thiết lập proxy cho npm.

npm sử dụng một file cấu hình và có thể thể thêm cấu hình qua lệnh npm config set ...

Thiết lập proxy

npm config set https-proxy http://[Your Proxy]:[Proxy Port]
npm config set proxy http://[Your Proxy]:[Proxy Port]

Bỏ thiết lập proxy

npm config delete https-proxy
npm config delete proxy

Xem thiết lập proxy

npm config get https-proxy
npm config get proxy

Xem tất cả các thiết lập

npm config list

Ví dụ

npm config set       proxy http://10.225.1.1:3128
npm config set https-proxy http://10.225.1.1:3128

Nó sẽ ghi ra file .npmrc các cấu hình (ví dụ C:\Users\nvhuyen\.npmrc).

Nếu vẫn chưa được, thử thêm 2 cấu hình sau (trên mạng nói thế, nhưng tôi không cần):

registry=http://registry.npmjs.org
strict-ssl=false
