Thiết kế giao diện Dark theme

Dark theme nhìn cool ngầu. Khi có dark theme thì có thể thêm hiệu ứng glowing, neon.

Nguyên lý:

  • Chủ yếu là đen (và các sắc thái)
  • Chữ trắng
  • Chỉ nên có 2 màu khác chủ đạo, nhiều nhất là 3
  • Icon nên đơn sắc, đồng nhất (có thể sử dụng filter grayscale)
  • Form control có viền thôi, màu nền vẫn là đen

Màu sắc

Body:
	background-color: #222529; /* Sắc đen 1 */
	color: #FFF; /* Màu 1, trắng */
Article:
	background-color: #2D3136; /* Sắc đen 2 */
Link
	color: #4ED8DA; /* Màu 2, xanh */
	color: #E06950; /* Màu 3, cam */		
	color: #E0B750; /* Màu 4, vàng */
Heading
	background-color: #393E44; /* Sắc đen 3 */
	color: #FFF; /* Màu 1, trắng */
