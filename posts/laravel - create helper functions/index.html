Tạo file Helpers.php trong thư mục app và sửa file composer.json như sau:

"autoload": {
    "classmap": [
        ...
    ],
    "psr-4": {
        "App\\": "app/"
    },
    "files": [
        "app/helpers.php" // <---- ADD THIS
    ]
},

Thực hiện lệnh:

composer dump-autoload

Bạn có thể gọi hàm định nghĩa trong Helpers.php ở bất cứ đâu (Controller, View, Route).
