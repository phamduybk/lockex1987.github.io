Cài đặt PHP trên Ubuntu

Chúng ta sẽ cài đặt PHP 7.2.

Xóa PHP 7.0 cũ và các gói của php-* nữa: 

sudo apt-get remove php php-*

Thêm repo và cài đặt: 

sudo add-apt-repository ppa:ondrej/php
sudo apt update
sudo apt install php7.2 php7.2-common php7.2-cli php7.2-fpm php7.2-mysql php7.2-gd php7.2-mbstring php7.2-xml php7.2-zip
