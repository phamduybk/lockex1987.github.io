Các nội dung dịch được khai báo trong các thẻ span với thuộc tính lang như sau:

<div>
    <span lang="en">Hello world!</span>
    <span lang="fr">Bonjour le monde!</span>
    <span lang="cn">你好，世界！</span>
</div>

Khai báo CSS để ẩn các ngôn ngữ không phải là mặc định:

<style>
    span[lang="fr"] {display: none}
    span[lang="cn"] {display: none}
</style>

Đổi ngôn ngữ bằng hàm switchLanguage, ví dụ:

<a href="#" onclick="switchLanguage('en')">English</a>
<a href="#" onclick="switchLanguage('fr')">French</a>
<a href="#" onclick="switchLanguage('cn')">Chinese</a>
