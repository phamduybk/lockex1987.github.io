Personal Website

Write your work so that you can do it easily in the second time. For example: how to install WordPress tutorial.

Write blog to create your own brand.

I want an online portfolio so that I can show my personal projects, my ideas. It will not be a tutorial-like website. There are so many tutorial-like websites out there, and some of them are really efficient.

I want a website with following features:

  • Users can comment on my project with a Google (Facebook) account
  • I will be notified when there is a new comment
  • I can customize it

But the website must have following criteria too:

  • Fast, high availability
  • Free

I have some options:

  • A free host (byethost) and build it from scratch in PHP
  • Google Blogger
  • WordPress
  • Facebook
  • Github

After some try, I finally decide to use Github. It just contains plain HTML files. I can run on local.
