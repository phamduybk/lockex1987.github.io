Đa ngôn ngữ với JavaScript

Sử dụng jquery-lang cho đơn giản.

https://github.com/Irrelon/jquery-lang-js

Bắt đầu

Thêm các thư viện cần thiết

  • jQuery
  • JS Cookie
  • jQuery Lang
<script src="lib/jquery/js/jquery.min.js"></script>
<script src="lib/js-cookie/js/js.cookie.js"></script>
<script src="lib/jquery-lang/js/jquery-lang.js"></script>

Thêm đoạn code sau vào head hoặc body (nên thêm vào head).

var lang = new Lang();
lang.dynamic('vi', 'lib/jquery-lang/langpack/vi.json');
lang.init({
    defaultLang: 'en'
});

Hàm init có thể có các tham số sau:

  • defaultLang
  • currentLang
  • cookie
  • allowCookieOverride

Load gói ngôn ngữ động (Nên dùng)

lang.dynamic('vi', 'lib/jquery-lang/langpack/vi.json');

Cấu trúc file ngôn ngữ như sau:

{
    // Define token (exact phrase) replacements here
    "token": {
        "Translate me": "Dịch tôi",
          "http://www.ioling.org/images/flags/uk.gif": "http://upload.wikimedia.org/wikipedia/commons/thumb/a/a9/Flag_of_Thailand.svg/1280px-Flag_of_Thailand.svg.png",
        "http://www.google.com/": "http://www.yahoo.com/",
        "lorem" : "Lorem Ipsum"
    },
    // Define regular expression replacements here
    "regex": [
        ["Budget", "งบประมาณ"],
        ["^Something at start of text", ""],
        ["This will be case insensitive", "i", ""]
    ]
}

Định nghĩa các đối tượng để dịch

Thêm thuộc tính lang vào các thẻ

Nó sẽ dịch:

  • Nội dung thẻ
  • Thuộc tính placeholder
  • Thuộc tính title
  • Thuộc tính alt
  • Thuộc tính href
  • Thuộc tính src

Thêm thuộc tính bằng cách:

Lang.prototype.attrList.push('data-name');

Bạn có thể sử dụng JavaScript:

window.lang.translate('Translate me')

Bạn có thể sử dụng data-lang-token với văn bản dài (key trong file JSON đỡ dài).

Dynamic Content

If you load content dynamically and add it to the DOM using jQuery the plugin will AUTOMATICALLY translate to the current language. Ensure you have added "lang" attributes to any HTML that you want translated BEFORE you add it to the DOM.

Switching languages

To switch languages simply call the .change() method, passing the two-letter language code to switch to. For instance here is a switcher that will change between English and Thai as used in the example page:

window.lang.change('vi')

The onclick event is the only part that matters, you can apply the onclick to any element you prefer.
