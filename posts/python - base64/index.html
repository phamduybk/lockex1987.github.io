Xử lý base64 với Python

Base64 thường được sử dụng để chuyển đổi qua lại giữa byte và xâu.

Để encode base64 trong Python sử dụng hàm base64.b64encode, để decode thì dùng hàm base64.b64decode.

Chú ý có sự khác biệt lớn giữa Python 2 và Python 3.

  • Trong Python 2:
    • Hàm base64.b64encode và base64.b64decode đều có tham số và giá trị trả về là xâu.
  • Trong Python 3:
    • Hàm base64.b64encode có tham số và giá trị trả về là mảng byte.
    • Hàm base64.b64decode có tham số là mảng byte hoặc xâu ASCII, giá trị trả về là mảng byte.

Như vậy, với Python 3 ta phải có thêm các bước biến đổi giữa mảng byte và xâu.

  • Sử dụng phương thức the_string.encode() của xâu để biến đổi thành mảng byte.
  • Sử dụng phương thức the_bytes.decode() của mảng byte để biến đổi thành xâu.
