Exception trong Python

Exception trong Python khá giống như trong Java. Cấu trúc của nó như sau:

try:
    # Statements
except Exception_1 as ex:
    # Process or pass
except Exception_2 as ex:
    # Process or pass
finally:
    # Optional
    # Always executed
    # Process or pass

Tại sao bạn cần đến Exception

Nếu bạn không xử lý Exception thì khi chương trình gặp Exception nó sẽ dừng lại luôn.

Nhiều khi cái bạn muốn đó là chương trình luôn trong tình trạng sống, ví dụ một chương trình chạy các job theo lịch chẳng hạn. Các job có thể lỗi ở một thời điểm nào đó, nhưng chương trình vẫn phải sống để chạy job tiếp theo.

def demo_1():
for i in range(10):
print(i)
x = 10 / 0

demo_1()

Đoạn code trên chỉ in được một lần i = 0 do gặp ZeroDivisionError: devision by zero.

Chúng ta chỉnh lại như sau:

def demo_2():
for i in range(10):
print(i)
try:
x = 10 / 0
except Exception as ex:
pass

demo_2()

Giờ thì chương trình sẽ chạy hết từ đầu đến cuối.
