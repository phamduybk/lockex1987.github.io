Import và export MySQL

Export

Export tất cả database:

mysqldump -u [user_name] -p[pass] --all-databases > all_db_backup.sql

Export một database (db_name) nào đó:

mysqldump -u [user_name] -p[pass] db_name > db_backup.sql

Chỉ định bảng cụ thể (table1, table2):

mysqldump -u [user_name] -p[pass] db_name table1 table2 > table_backup.sql

Tự động nén đầu ra với gzip (nếu dữ liệu lớn):

mysqldump -u [user_name] -p[pass] db_name | gzip > db_backup.sql.gz

Export từ remote server:

mysqldump -P 3306 -h [ip_address] -u [user_name] -p[pass] db_name > db_backup.sql

Import

Chú ý bạn phải tạo DB trước:

CREATE DATABASE db_name CHARACTER SET=utf8 COLLATE=utf8_vietnamese_ci;

Câu lệnh:

mysql -u [user_name] -p[pass] db_name < db_backup.sql

Import vào remote server:

mysql -u [user_name] -p[pass] -h [ip_address] db_name < db_backup.sql
