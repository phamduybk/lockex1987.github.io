Noti

Đây là một thư viện JavaScript cho việc thông báo, thay thế cho hai hàm JavaScript thuần là alert và confirm.

Thay cho alert, chúng ta chia thành 4 loại sau:

  • info
  • error
  • success
  • warning

Thư viện này không phức tạp hóa vấn đề lên, mà chỉ là bổ sung thêm cho hai hàm alert và comfirm về mặt giao diện: giao diện thống nhất với ứng dụng, thống nhất giữa các trình duyệt, có tiếng Việt,...

Cách sử dụng

<link rel="stylesheet" href="https://lockex1987.github.io/posts/project%20-%20noti/css/noti.css" />

<script src="https://lockex1987.github.io/posts/project%20-%20noti/js/noti.js"></script>

<script>
    noti.error("Bạn chưa được phân quyền thực hiện chức năng này!");
    
    noti.success('Thêm mới thành công người dùng', { autoClose: false });
    
    noti.info('Hệ thống đang tạm dừng. Vui lòng quay lại sau.');
    
    noti.warning('Hãy chắc chắn bạn có đẩy đủ quyền thực hiện');
    
    noti.confirm('Bạn có muốn xóa bản ghi?', function () {
        noti.success('Xóa thành công');
    });
    
    noti.openModal('myModal');
    
    noti.closeModal('myModal');
</script>

Tính năng

Các tính năng hiện có:

  • Top center
  • Click close
  • Auto close or sticky
  • Transparent background

Các tính năng có thể làm thêm:

  • Modal dialog
  • Responsive
  • Animated
  • x Close
  • Inline alert (phía trên một đối tượng nào đó)
  • Web Notification API
  • Click Through
  • HTML content
