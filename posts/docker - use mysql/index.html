Sử dụng MySQL với Docker

Có 2 bản MySQL trên Docker Hub.

  • Một bản là official, địa chỉ ở https://hub.docker.com/_/mysql/ .
  • Một bản thì lại có owner là chính MySQL, địa chỉ ở https://hub.docker.com/r/mysql/mysql-server/ .

Chúng ta hãy thử với bản thứ 2 (không phải official, nhưng owner là MySQL) trước.

Pull image: 

docker pull mysql/mysql-server

Chạy: 

docker run --name=mysql-1 -d mysql/mysql-server

Có thể thêm tham số cổng: 

docker run --name=mysql-1 -d -p 3308:3306 mysql/mysql-server

Để cổng 3308 để tránh trùng.

Lấy password tự sinh: 

docker logs mysql-1 2>&1 | grep GENERATED

Truy cập thử và đổi password: 

docker exec -it mysql-1 mysql -uroot -p

Đổi password trong MySQL: 

ALTER USER 'root'@'localhost' IDENTIFIED BY '123456';

Các thao tác liên quan đến container: 

docker stop mysql-1
docker start mysql-1
docker restart mysql-1
docker rm mysql-1
docker exec -it mysql-1 bash
