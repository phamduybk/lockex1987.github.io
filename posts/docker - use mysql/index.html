Pull image:

docker pull mysql/mysql-server

Chạy:

docker run --name=mysql1 -d mysql/mysql-server

Có thể thêm tham số cổng:

docker run --name=mysql1 -d -p 3308:3306 mysql/mysql-server

Để cổng 3308 để tránh trùng.

Lấy password tự sinh:

docker logs mysql1 2>&1 | grep GENERATED

Truy cập thử và đổi password:

docker exec -it mysql1 mysql -uroot -p

Đổi password trong MySQL:

ALTER USER 'root'@'localhost' IDENTIFIED BY 'newpassword';

Các thao tác liên quan đến container:

docker stop mysql1
docker start mysql1
docker restart mysql1
docker rm mysql1
docker exec -it mysql1 bash

This image can also be used as a client for non-Docker or remote MySQL instances:

docker run -it --rm mysql mysql -hsome.mysql.host -usome-mysql-user -p
