Pull image:

docker pull mysql

Chạy:

docker run --name some-mysql -e MYSQL_ROOT_PASSWORD=my-secret-pw -p 3306:3306 -d mysql

Truy cập thử

docker exec -it some-mysql mysql -uroot -p

Nhập password là:

my-secret-pw
