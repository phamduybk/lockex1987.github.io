Sửa đường dẫn path

Cấu hình ở file /etc/profile

JAVA_HOME=/opt/jdk-9.0.4
M2_HOME=/home/locke/program/maven
NODE_HOME=/home/locke/program/node
PATH=$PATH:$JAVA_HOME/bin:$M2_HOME/bin:$NODE_HOME/bin

Phải khởi động lại thì mới ăn.

Chúng ta cũng có thể sửa file ~/.bashrc hoặc ~/.profile hoặc ~/.bash_profile. Khi đó thay chỉ có hiệu lực với user hiện tại.

File ~/.profile sẽ không được đọc nếu ~/.bash_profile tồn tại.

PATH="$HOME/.composer/vendor/bin:$HOME/bin:$HOME/.local/bin:$PATH"
