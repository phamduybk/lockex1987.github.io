CTTD

What is CTTD? It's a Java library that contains almost common patterns.

Diagram:

  • CTTD base on third party library like apache commons, mail. You can think of it as a wrapper.
  • CTTD contains common patterns (use cases)
  • With each pattern, we have web features (in Universal Website) and utilities

Example, we have rename utility, we have thumbnail web features and have batch resize utility.

Common Patterns:

  • compress file: handle compressed files, save disk space
  • cryptography: encrypt sensitive information
  • download (parse XML, service API)
  • email (send mail, process incoming mail?)
  • excel (import, export)
  • file manager (CRUD)
  • json (parse, output)
  • pdf & doc (export)
  • regular expression (filter, validate)
  • scheduler (automatic)
  • sync (synchronize databases, files?)
  • upload (thumbnail)
  • Unicode internationalization
