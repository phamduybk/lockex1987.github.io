3. Common tags

Stt Tag Mục đích Attribute Ý nghĩa Ghi chú
1 attachmentFile Đính kèm file, hiện các file đã đính kèm, cho phép download và xóa các file đã đính kèm. viewOnly Chỉ được phép xem, không được phép thêm, xóa Các attribute, thuộc tính của form phải đặt tên đúng chuẩn
enableDelete Có được phép xóa các file đính kèm cũ không
2 button Tạo nút thực hiện một hành động nào đó.      
3 dataPicker Chọn một đối tượng.      
4 dateChooser Chọn ngày tháng. name Sinh ra truong name va id  
id Neu chi dinh se sinh ra truong id
styleClass Lop CSS de validate
value Gia tri
5 multiDataPicker Chọn đối tượng, cho phép chọn nhiều, loại bỏ các đối tượng đã chọn. dataPickerId Id của data picker. Cần khai báo một trường Long[] arrHidden<dataPickerId> để lấy dữ liệu về  
object Bảng hoặc view trong CSDL
firstLabel Nhãn của trường tìm kiếm đầu tiên

firstField Trường tìm kiếm đầu tiên
secondField Trường tìm kiếm thứ hai
selectField Trường chọn, thường là trường ID của bảng
filterCondition Điều kiện lọc của bảng và view
orderField Trường sắp xếp
6 link Tạo link để thực hiện một hành động nào đó.      
7 dialog (popupChooser) Mở một màn hình mới để thực hiện thao tác nào đó.      
8 navigator Switch giữa màn hình form và màn hình danh sách.      
9 tree Cây.      
10 treeChooser Chọn đối tượng dạng cây. treeChooserId ID của điều khiển chọn cây
