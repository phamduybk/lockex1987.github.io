My Framework Documentation

Mục đích của tài liệu này ngoài việc hướng dẫn cách sử dụng framework chung của TTPM còn hướng nhân viên phát triển tới một phong cách lập trình thống nhất, có tổ chức.
Tài liệu được chia thành các phần sau:
  • Giới thiệu chung
  • Bắt đầu một dự án mới
  • Các coding pattern điển hình
  • Common tags
  • Các lớp tiện ích Java
  • Các tiện ích Javascript
  • Các lỗi thường gặp
  • Giao diện chương trình
  • Hướng phát triển tiếp theo
  • Tùy biến framework
  • Hướng dẫn làm prototype
  • Triển khai ứng dụng
Hai phần đầu có mục đích giúp nhân viên mới vừa tìm hiểu vừa thực hành.
Các phần sau có mục đích tra cứu.
Phần cuối cùng dành cho dev có kinh nghiệm để bắt đầu một dự án mới.
