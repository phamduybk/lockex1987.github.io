8. Hướng dẫn build

Điều kiện tiên quyết:
  • Lấy code từ SVN
  • New project từ Netbeans (chú ý sửa context path, bỏ "Deploy on save", bỏ share,...)
  • Riêng VPM phải thêm project viettelLibrary
Hướng dẫn build:
  • Lấy hết code mới từ SVN
  • Clean & Build ở Netbeans để ra file war
  • Dùng WinSCP, putty để đăng nhập
  • Di chuyển đến thư mục bin của Tomcat
  • Tắt Tomcat
    ./shutdown.sh (tắt bình thường)
    ps -ef | grep `pwd` (kiểm tra xem còn tiến trình hay không)
    kill -9 <PID> (kill tiến trình nếu tắt bình thường không được)
  • Xóa thư mục triển khai, log cũ, cache
    rm -rdf ../webapps/*
    rm -rdf ../logs/*
    rm -rdf ../work/*
  • Đẩy file war lên, đổi lại tên cho đúng (nếu cần)
  • Start Tomcat
    ./startup.sh
  • Chỉnh file cấu hình
    File vpsclient, cas, config, hibernate,... (tùy dự án)
  • Shutdown và startup lại Tomcat
    ./shutdown.sh
    ./startup.sh
  • Đăng nhập kiểm tra
