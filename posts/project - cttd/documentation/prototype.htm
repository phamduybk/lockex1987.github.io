Hướng dẫn làm prototype

Prototype framework giúp việc làm prototype nhanh hơn, hiệu quả hơn, chi tiết hơn, thực tế hơn.
Dev cũng có thể sử dụng lại prototype.
Đặc điểm của Prototype framework là chạy không cần CSDL, không cần Form, Controller, DAO.

Bước 1: Khai báo ở struts-config.xml

Tìm đến action Authenticate, thêm một trang trả về dạng:
<forward name="report" path="report" />
Chú ý name, path phải là duy nhất.

Bước 2: Khai báo ở tiles-defs.xml

Tạo một definition trùng với khai báo ở struts-config.xml:
<definition name="report" extends="base.definition">
    <put name="title" value="Báo cáo" />
    <put name="body" value="/WEB-INF/jsp/report/report.jsp" />
</definition>

Bước 3: Tạo file JSP

Tạo file JSP trùng tên, ở cùng thư mục như khai báo trong tiles-defs.xml.

Bước 3: Tạo menu

Tạo một menu ở file baseLayout.jsp có dạng:
<li>
<a href="${contextPath}/authenticateAction.do?<commonTags:encrypt text="className=admin.controller.AuthenticateController&methodName=actionGotoPage&pageForward=report" />">Báo cáo</a>
</li>



OK, vậy là chúng ta có thể làm prototype trên file JSP vừa tạo ra.
