File filter

Lọc ra các file để có thể upcode nhanh.

Sử dụng lệnh git status để lấy ra danh sách các file thay đổi.

Copy danh sách các file thay đổi vào file changes.txt.

Sau đó thực hiện lệnh sau:

java -jar fileFilter.jar <path to source folder>

Các file thay đổi sẽ được lọc và copy vào thư mục filtered. Cấu trúc thư mục được giữ nguyên nên khi upcode chúng ta chỉ việc copy nguyên cả thư mục.
