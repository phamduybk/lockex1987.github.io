Dùng để viết các API đơn giản.

Tạo một project đơn giản

Chỉ gồm 3 bước đơn giản.

Bước 1: Tạo một Maven project và thêm dependency sau vào file pom.xml:

<dependency>
    <groupId>com.sparkjava</groupId>
    <artifactId>spark-core</artifactId>
    <version>2.7.2</version>
</dependency>

Bước 2: Tạo 1 class và bắt đầu code:

import static spark.Spark.*;

public class HelloWorld {
    public static void main(String[] args) {
        get("/hello", (req, res) -> "Hello World");
    }
}

Bước 3: Chạy và kiểm tra ở URL sau:

http://localhost:4567/hello
