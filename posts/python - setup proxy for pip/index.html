Cấu hình proxy cho pip

pip là một tiện ích để cài các thư viện từ mạng của Python. Nó gần tương tự như Maven của Java.

Ở môi trường mà không có kết nối Internet trực tiếp và chỉ có thể kết nối qua proxy (ví dụ như ở công ty), chúng ta vẫn có kể cài các thư viện với pip bằng cách thêm tùy chọn --proxy như sau:

$ pip install <library> --proxy=<proxy server and port>

Ví dụ:

$ pip install PyMySQL --proxy="proxy.mydomain.vn:3128"
