Không tương thích giữa Chrome và Firefox

Những extension mà được xây dựng từ WebExtension API được thiết kế sao cho tương thích với Chrome và Opera. Những extension mà viết cho Chrome và Opera sẽ chạy trên Firefox với ít các thay đổi nhất.

Tuy nhiên, Firefox hiện tại chỉ hỗ trợ một số các tính năng và API mà được hỗ trợ bởi Chrome và Opera.

Hàm callback và chrome.* namespace

Ở Chrome, extension truy cập các API sử dụng chrome namespace:

chrome.browserAction.setIcon({ path: "path/to/icon.png" });

Firefox truy cập thông qua browser namespace:

browser.browserAction.setIcon({ path: "path/to/icon.png" });

Nhiều API là không đồng bộ. Ở Chrome, các API không đồng bộ sử dụng hàm callback, và runtime.lastError để báo lỗi:

Firefox dùng promise.

Firefox hỗ trợ cả chrome và callback lẫn browser và promise. Điều đó có nghĩa là nhiều Chrome extension sẽ chạy ở Firefox mà không phải thay đổi gì.
