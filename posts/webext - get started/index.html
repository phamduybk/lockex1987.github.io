Web Extension

Web extensions allow developers to extend and modify the functionality of web browsers. They are written using standard Web technologies - JavaScript, HTML, CSS - plus some dedicated JavaScript APIs.

An extension could:

  • Change the  appearance or content of particular websites
  • Modify the user interface
  • Add new features to the browser
  • (Imagination of people is limitless)

I mostly use Firefox with some extensions: Firebug, Scrapbook, Adblock, DownloadThemAll!, Download YouTube Videos as MP4.

There are currently several toolsets for developing (Firefox) web extensions, but WebExtensions will become the standard by the end of 2017. The rest are expected to be deprecated over the same period of time.

Add-on SDK, XUL and XPCOM-based are other toolsets. Just like Node JS, everything is changing very fast. You should use the official documentation on home pages. Personal web tutorials become out-of-dated very soon.

There are web extensions for Chrome, Safari too.

There are other types of web extensions:

  • Lightweight themes
  • Mobile
  • Search engine plugins
  • User dictionaries
  • Language pack
