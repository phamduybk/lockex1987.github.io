Lấy tham số từ dòng lệnh trong Python

Để chạy được chương trình Python, bạn phải thực hiện từ cửa sổ dòng lệnh với cú pháp kiểu như sau:

$ python my_file.py var1 var2 var3

Để lấy được các tham số var1, var2, var3,... chúng ta sử dụng danh sách sys.argv.

import sys

print('Type:', type(sys.argv))
print(sys.argv)

Ví dụ chúng ta chạy lệnh sau:

$ python my_file.py one two "has spaces"

Thì kết quả sẽ là:

Type: <class 'list'>
['my_file.py', 'one', 'two', 'has spaces']

Chú ý phần tử đầu tiên của danh sách sys.argv là tên file. Do đó để lấy các tham số thì chúng ta bắt đầu lấy từ phần tử thứ hai.
