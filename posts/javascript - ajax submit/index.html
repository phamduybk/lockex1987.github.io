AJAX Submit

Features

  • Multiple file upload
  • Drag and Drop support
  • Progress bar
  • File information
  • Nice GUI
  • Image preview
  • Cancel, Remove options
  • File type, file size validation
  • With or without submit (if it's an application focuses on uploading, you should submit automatically, instantly)

No Grail, one thing to do them all.

With this simple situation, you should make your own function from scratch. It will be expanded in the future.

There are three subjects (knowledge) you have to know:

  • The XMLHttpRequest object
  • The FileReader object from the new File API
  • The FormData object from XMLHttpRequest2

VPM project uses jQuery Form plugin (and you should use this too with similar projects).

Multiple file

No native form element. A file input with multiple option only allows choose multiple files in the same directory.

The name of the file input must has brackets at the end of it. Server will recognize it as an array.
