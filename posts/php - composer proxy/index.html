Thiết lập proxy cho shell:

Windows

set http_proxy=http://proxy.cyberspace.vn:3128
set https_proxy=http://proxy.cyberspace.vn:3128
echo %http_proxy%
echo %https_proxy%

Linux

export http_proxy=http://proxy.cyberspace.vn:3128
export https_proxy=http://proxy.cyberspace.vn:3128
echo $http_proxy
echo $https_proxy
