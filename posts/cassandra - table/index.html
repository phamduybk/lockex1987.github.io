Cassandra Table

Tạo bảng

create table if not exists student (id int, name text, email text, phone text, primary key(id));

Chỉnh sửa bảng

Ví dụ thêm class vào bảng student đã tạo sử dụng cú pháp:

alter table student add address text;

Chèn và cập nhật dữ liệu trong bảng

Chèn dữ liệu sử dụng cú pháp sau:

insert into student(id, address, email, name, phone) values(1, 'Vietnam', 'abc@xxx.com', 'ABC', '09999999');

Cập nhật dữ liệu:

update student set email = 'ABC@gmail.com'  where id = 1;

Truy vấn dữ liệu

Để truy vấn dữ liệu sử dụng lệnh SELECT

select * from student;
