Off-Canvas Menu

Một demo nhỏ bao gồm các kỹ thuật sau:

  • Menu ở bên phải
  • Nút hiện menu
  • Nút ẩn menu
  • Khi click vào link thì ẩn menu
  • Click ra ngoài menu thì ẩn menu
  • Animation đơn giản với CSS
  • Code tổng quát, nếu thêm các thẻ div khác sẽ tự động nhận sự kiện
  • Sử dụng HTML Entity, không sử dụng ảnh hay Font Awesome
