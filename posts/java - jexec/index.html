jexec

Có nhiều trường hợp, chúng ta có 1 đoạn code Java mẫu và chúng ta muốn nó thực hiện luôn 1 cách đơn giản.

Chúng ta không muốn bật các IDE cồng kềnh như Eclipse, NetBeans và tạo project.

Chúng ta chỉ cần 1 file Java, 1 trình soạn thảo đơn giản và cửa sổ dòng lệnh.

Nhưng chúng ta vẫn phải đánh các lệnh javac để dịch lại mỗi khi có thay đổi.

Sửa file .bashrc (thường ở thư mục home của người dùng), thêm 1 hàm mới như sau:

# Java compile and run
jexec() {
    fileName=$1
    javac -encoding utf8 "$fileName.java"
    java "$fileName"
}

Chúng ta thực hiện lệnh sau, vừa dịch vừa chạy:

jexec JexecDemo
