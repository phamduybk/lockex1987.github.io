insertAdjacentHTML

If we want to insert content in a HTML document we have three ways to do it:

  • Using DOM methods like createNode and appendChild
  • Using Document Fragments
  • Using innerHTML

One can arguably say we also have document.write for few use cases.

Không biết cái Firefox add-on reviewer có cho phép hàm này không? (Hàm innerHTML không cho phép).

Cú pháp:

element.insertAdjacentHTML(position, text);

position

A DOMString representing the position relative to the element; must be one of the following strings:

  • beforebegin: Before the element itself.
  • afterbegin: Just inside the element, before its first child.
  • beforeend: Just inside the element, after its last child.
  • afterend: After the element itself.

text

text is the string to be parsed as HTML or XML and inserted into the tree.

Vị trí:

<!-- beforebegin -->
<p>
    <!-- afterbegin -->
    foo
    <!-- beforeend -->
</p>
<!-- afterend -->

Ví dụ:

This method inserts a specified text as HTML, into a specified position in the document.
