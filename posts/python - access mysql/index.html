Truy cập MySQL với Python

Chúng ta sẽ thực hành truy cập MySQL với thư viện pyMySQL trong Python.

Cài đặt thư viện như sau:

$ pip install PyMySQL

Mở kết nối, trả về đối tượng Connection.

Tiếp theo là tạo đối tượng cursor.

Sau đó thì thực hiện các câu lệnh bằng cursor.execute.

Tạo một bảng để test trong MySQL như sau:

CREATE TABLE pymysql_demo (
    id int(10) AUTO_INCREMENT,
    full_name varchar(100),
    access_time datetime,
    status int(1),
    PRIMARY KEY (id)
) AUTO_INCREMENT=1 DEFAULT CHARSET=utf8;
