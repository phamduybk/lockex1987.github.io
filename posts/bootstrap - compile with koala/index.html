Biên dịch Bootstrap với Koala

Mục đích:

  • Để hiểu Bootstrap làm như thế nào
  • Để chỉnh sửa Bootstrap

Download project từ GitHub.

Chỉ giữ lại thư mục scss.

Có thể xóa tiếp các file bootstrap-grid.scss, bootstrap-reboot.scss.

Download, cài đặt, mở app Koala, thêm thư mục.

Cấu hình Koala dịch từ thư mục scss và output ra thư mục css ở file koala-config.json.

{
    "mappings": [
        {
            "src": "scss",
            "dest": "css"
        }
    ]
}
