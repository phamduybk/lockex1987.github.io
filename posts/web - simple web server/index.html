Nhiều khi bạn cần một web server thực sự:

  • Gọi AJAX sẽ không thể thực hiện ở file:// do lỗi bảo mật
  • Mock dữ liệu trả về từ server

Có nhiều cách thực hiện. Chúng ta sẽ tạo một web server đơn giản ở địa chỉ http://localhost:8080 từ thư mục nào đó.

Sử dụng Python

Python 2

python -m SimpleHTTPServer 8080

Python 3

python -m http.server 8080

Sử dụng PHP built-in web server

php -S localhost:8080

Để truy cập được từ máy khác thì thay localhost bằng 0.0.0.0.

php -S 0.0.0.0:8080

Sử dụng Node

Dùng package http-server.

Cài đặt

npm install http-server -g

Sử dụng

http-server

Sử dụng PHP, chúng ta có thể mock dữ liệu đơn giản.

test_get.php

test_json.php

test_post.php

test_upload.php

Các lệnh curl

curl localhost:8080/test_get.php

curl --request POST localhost:8080/test_get.php

curl --request DELETE localhost:8080/test_get.php

curl --request PUT localhost:8080/test_get.php

curl -I localhost:8080/test_get.php

curl -X POST localhost:8080/test_post.php -d 'username=root&password=abc123'
