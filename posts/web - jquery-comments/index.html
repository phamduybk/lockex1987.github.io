jQuery Comment

Comment làm một chức năng phổ biến. Nhưng làm thế nào để có thể sử dụng được lại code.

Sử dụng thư viện jquery-comments. jquery-comments cung cấp các chức năng UI và gắn cho chúng các hàm callback mà bạn muốn làm với dữ liệu. Thư viện có nhiều cấu hình, dễ tích hợp.

Tích hợp

Thêm thư viện và các phụ thuộc như sau:

<!-- Font Awesome (phụ thuộc không bắt buộc) -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/font-awesome/4.5.0/css/font-awesome.min.css">

<!-- jQuery (phụ thuộc bắt buộc, >= 1.9.0) -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/1.9.0/jquery.min.js"></script>

<!-- Text complete (phụ thuộc không bắt buộc) -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery.textcomplete/1.8.0/jquery.textcomplete.js"></script>

<!-- Thư viện chính jquery-comment -->
<link rel="stylesheet" href="css/jquery-comments.css">
<script src="js/jquery-comments.js"></script>

Thêm một vùng DIV

<div id="comments-container"></div>

Khởi tạo JavaScript

$('#comments-container').comments({
    profilePictureURL: 'https://app.viima.com/static/media/user_profiles/user-icon.png',
    getComments: function(success, error) {
        var commentsArray = [{
            id: 1,
            created: '2015-10-01',
            content: 'Lorem ipsum dolort sit amet',
            fullname: 'Simon Powell',
            upvote_count: 2,
            user_has_upvoted: false
        }];
        success(commentsArray);
    }
});
