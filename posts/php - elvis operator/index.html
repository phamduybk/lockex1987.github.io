It evaluates to the left operand if the left operand is truthy, and the right operand otherwise.

In pseudocode,

foo = bar_1 ?: bar_2;

roughly resolves to

foo = bar_1 ? bar_1 : bar_2;

or

if (bar_1) {
    foo = bar_1;
} else {
    foo = bar_2;
}

with the difference that bar_1 will only be evaluated once.

Some more examples:

<?php
    var_dump(5 ?: 0); // 5
    var_dump(false ?: 0); // 0
    var_dump(null ?: 'foo'); // 'foo'
    var_dump(true ?: 123); // true
    var_dump('rock' ?: 'roll'); // 'rock'
?>

By the way, it's called the Elvis operator.
