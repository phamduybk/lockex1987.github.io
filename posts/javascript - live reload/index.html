Một web extension tự động cập nhật phiên bản mới nhất của HTML, JS, CSS. Bạn không phải nhấn F5 thủ công.

Demo với live.js

Không hỗ trợ file local, phải trên 1 web server nào đó.

Web server đơn giản:

php -S localhost:8080

Kiểm tra:

localhost:8080
