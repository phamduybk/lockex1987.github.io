Replace OpenJDK with OracleJDK on Linux

You can completely remove the OpenJDK and fresh install Oracle Java by the following steps:

Check current version and remove OpenJDK completely

java -version
which java
ls /usr/bin/java*
sudo update-alternatives --config java
sudo apt-get purge openjdk*
sudo apt-get autoremove

Download JDK

Next download JDK (or JRE). If the server has Internet connection, you can use wget command to download JDK directly:

sudo -iu root
cd /opt/
wget --no-cookies --no-check-certificate --header "Cookie: gpw_e24=http%3A%2F%2Fwww.oracle.com%2F; oraclelicense=accept-securebackup-cookie" "http://download.oracle.com/otn-pub/java/jdk/8u101-b13/jdk-8u101-linux-x64.tar.gz"
tar xzvf jdk-8u101-linux-x64.tar.gz

The official download link from Oracle.

Make sure to download appropriate file, eg: jdk-8u51-linux-x64.tar.gz.

wget --no-cookies --no-check-certificate --header "Cookie: oraclelicense=accept-securebackup-cookie" http://download.oracle.com/otn-pub/java/jdk/8u112-b15/jdk-8u112-linux-x64.rpm
wget --no-cookies --no-check-certificate --header "Cookie: oraclelicense=accept-securebackup-cookie" http://download.oracle.com/otn-pub/java/jdk/8u112-b15/jdk-8u112-linux-x64.tar.gz

In all cases above, subst 'i586' for 'x64' to download the 32-bit build.

Install

After extracting the zip file, use alternatives command to install.

alternatives --install /usr/bin/java java /opt/jdk1.8.0_101/bin/java 1
alternatives --config java
java -version

Use similar commands with javac and javaws.

Tell the system that the new Oracle Java version is available 

sudo update-alternatives --install "/usr/bin/java" "java" "/opt/jdk1.8.0_101/bin/java" 1
sudo update-alternatives --install "/usr/bin/javac" "javac" "/opt/jdk1.8.0_101/bin/javac" 1
sudo update-alternatives --install "/usr/bin/javaws" "javaws" "/opt/jdk1.8.0_101/bin/javaws" 1
sudo update-alternatives --set java /opt/jdk1.8.0_101/bin/java
sudo update-alternatives --set javac /opt/jdk1.8.0_101/bin/javac
sudo update-alternatives --set javaws /opt/jdk1.8.0_101/bin/javaws

After install successfully, you can delete the zip file:

rm /opt/jdk-8u101-linux-x64.tar.gz

Update system variables

Edit file /etc/profile (sudo gedit /etc/profile), add following lines at the end of file:

JAVA_HOME=/opt/jdk1.8.0_101
PATH=$PATH:$JAVA_HOME/bin

export JAVA_HOME
export PATH

That's all.
