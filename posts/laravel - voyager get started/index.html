Giới thiệu Voyager

Có những chức năng làm đi làm lại nhiều lần.

Những chức năng phần admin chủ yếu là thêm - sửa - xóa dữ liệu.

Những chức năng phần admin thì không cần giao diện đặc biệt.

Bắt đầu

Tạo project Laravel mới:

composer create-project --prefer-dist laravel/laravel my-admin

Thêm package voyager:

composer require tcg/voyager

Tiếp theo, chỉnh sửa cấu hình kết nối đến DB trong file .env:

DB_HOST=localhost
DB_DATABASE=homestead
DB_USERNAME=homestead
DB_PASSWORD=secret

Cập nhật URL của trang web với cấu hình APP_URL trong file .env:

APP_URL=http://localhost:8000

Cuối cùng, chúng ta cài đặt Voyager. Bạn có thể cài đặt cùng với việc khởi tạo dữ liệu hoặc không. Dữ liệu khởi tạo sẽ bao gồm 1 tài khoản admin, 1 demo page, 4 demo post, 2 categorie và 7 setting.

Cài đặt Voyager không có dữ liệu:

php artisan voyager:install

Cài đặt Voyager có dữ liệu luôn:

php artisan voyager:install --with-dummy

Start server:

php artisan serve

Mở trình duyệt kiểm tra:

http://localhost:8000/admin

Tạo tài khoản admin

Nếu bạn cài đặt Voyager với dữ liệu khởi tạo, bạn có thể đăng nhập với thông tin như sau:

  • email: admin@admin.com
  • password: password

Chú ý: user chỉ được tạo nếu không có user nào trong database.

Bạn có thể gán quyền admin với người dùng đang có:

php artisan voyager:admin your@email.com

Bạn cũng có thể tạo mới user với tùy chọn --create.

php artisan voyager:admin your@email.com --create

Bạn sẽ được nhắc nhập tên và password.
