Xử lý JSON với Python

JSON là một trong những định dạng trao đổi dữ liệu thông dụng nhất hiện nay. Để xử lý JSON với Java, chúng ta thường phải thêm các thư viện ngoài như GSON. Ngoài ra chúng ta cũng phải định nghĩa các class tương ứng với cấu trúc dữ liệu JSON khá là dài dòng.

Xử lý JSON với Python rất là tự nhiên và dễ dàng.

Để chuyển dữ liệu Python về xâu JSON chúng ta sử dụng hàm json.dumps, ngược lại sử dụng hàm json.loads.

import json

raw_json = '''
{
"field_str": "Hello Python",
"field_int": 123,
"field_float": 45.6,
"field_array": [1, 2, 3, 4, 5]
}
'''
data = json.loads(raw_json)
string_json = json.dumps(data)
pretty_json = json.dumps(data, indent = 2)

print(data)
print(data["field_str"])
print(string_json)
print(pretty_json)

Để xâu JSON đẹp chúng ta có thể thêm tham số indent cho hàm json.dumps.
