Xin chào các bạn. Bài viết tiếp theo đó là cách cài đặt Docker trên Windows. Đây là hướng dẫn step by step được thực hiện trên Windows 10.

Download Docker & Start setup

Đầu tiên các bạn cần tải file cài đặt mới nhất tại đây

Double click vào file cài đặt vừa tải về và trình cài đặt sẽ tự động giải nén các gói cần thiết. Trong quá trình cài đặt có thể Docker sẽ yêu cầu bật Hyper-V, bạn chỉ cần OK trình cài đặt sẽ tự động làm việc này cho bạn. Toàn bộ quá trình cài đặt được diễn ra hoàn toàn tự động và có thể sẽ restart máy tính vài lần.

Up & Run

Sau khi cài đặt thành công, hãy tìm Docker ở start menu và chạy Docker. Lần đầu tiên chạy có thể mất một chút thời gian.

Docker sẽ chạy ngầm ở thanh taskbar, vì vậy hãy kiểm tra xem Docker đã thật sự chạy chưa nhé.

Bước tiếp theo, chúng ta sẽ chạy chương trình Docker đầu tiên để xác định Docker đã được cài đặt chính xác theo các bước sau nhé:

Bước 1: Mở Powershell

Bước 2: Nhập lệnh và bấm Enter

docker run hello-world

Bước 3: Kết quả đạt được như hình dưới đây bạn đã thành công.
