Redis khó cài đặt trên Windows.

Sử dụng Docker.

Pull image Redis:

docker pull redis

Kiểm tra:

docker images

Start Redis container:

docker run -p 6379:6379 redis

Lấy IP của máy ảo:

docker-machine ip

Giả sử kết quả là 192.168.99.100.

Kết nối từ máy host (máy Windows bình thường):

redis-cli -h 192.168.99.100
